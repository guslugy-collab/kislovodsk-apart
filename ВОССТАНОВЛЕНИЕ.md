# ВОССТАНОВЛЕНИЕ — если потерян компьютер / нужен проект с нуля

Проект — статический сайт (один `index.html`), всё хранится в GitHub. Секретов нет.

## На новом компьютере
1. Установить **Git** и браузер.
2. Войти в GitHub (аккаунт `guslugy-collab`).
3. Склонировать репозиторий:
   ```
   git clone https://github.com/guslugy-collab/kislovodsk-apart.git
   ```
4. Открыть в папке `НАЧНИ-ОТСЮДА.md` и следовать ему.
5. Запуск — просто открыть `index.html` в браузере. Ничего устанавливать не нужно (нет зависимостей, нет `.env`).

## Живой сайт (демо)
https://guslugy-collab.github.io/kislovodsk-apart/ — обновляется автоматически при `git push` (GitHub Pages).

## Откат к версии
`git tag` — список версий. Вернуться: `git checkout M1` (или назвать нужный тег).

## Секреты
Их нет. Если позже появятся (напр. токен формы) — только в `.env` (он в `.gitignore`), копию положить в Google Диск, в git НЕ коммитить.
